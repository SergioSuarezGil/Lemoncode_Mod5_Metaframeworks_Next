## Desafíos implementados

- Listado de casas rurales (SSG/ISR)
- Detalle de casa rural (SSG/SSR)
- Búsqueda por nombre/ubicación
- Botón de reservar
- Optimización de imágenes con Next.js
- Estructura modular y reutilizable
- Estilos SASS + BEM + Material Tailwind
- Linting y Prettier configurados
- A mayores he creado un HouseReviewSchema porque lo hacemos mucho en mi empresa para mapear la resta de la API.
- Y tambien he añadido un script que arranca api y front en paralelo para facilitar la ejecución del proyecto.
